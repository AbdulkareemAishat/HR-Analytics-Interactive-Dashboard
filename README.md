# HR-Analytics-Interactive-Dashboard
## Project Overview 
An interactive HR Analytics Dashboard built in Microsoft Excel to visualize employee demographics, job satisfaction, income and workforce trends using Pivot tables, Pivot charts, slicers, and KPI cards for data-driven HR decision making.


## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
  
## Key Insights
- Job satisfaction by Age groups
- Monthly income by Job role
- Job involvements by Education field
- Top 5 Overtime workers by Job role
- Performance by Marital status
- Monthly income by Department
